# Test Goal

Validate that users can browse products from the home page.

# Requirement

User can browse products from the home page.

# Acceptance Criteria

1. Product cards are visible on first load.
2. Product name and price are visible on each card.
3. User can scroll and continue viewing more products.
4. Product grid remains interactive while browsing.
