# Test Goal

Validate order confirmation is shown after successful checkout.

# Requirement

User sees order confirmation after successful checkout.

# Acceptance Criteria

1. Successful checkout routes to confirmation state.
2. Confirmation message is visible to user.
3. Order reference or summary is displayed when available.
4. User can identify the purchase as completed.
