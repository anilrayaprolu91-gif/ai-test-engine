# Test Goal

Validate successful guest checkout with valid required fields.

# Requirement

User can proceed through guest checkout with valid required fields.

# Acceptance Criteria

1. User can proceed from cart to checkout.
2. Required fields accept valid input.
3. User can submit the checkout form.
4. Checkout completes without validation errors.
