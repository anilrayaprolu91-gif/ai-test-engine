# Test Goal

Validate adding products to cart from listing and detail views.

# Requirement

User can add a product to cart from listing and detail page.

# Acceptance Criteria

1. Add-to-cart from listing succeeds.
2. Add-to-cart from detail page succeeds.
3. Cart count or indicator updates after add.
4. Added product appears in cart summary.
