# Test Goal

Validate product sorting options.

# Requirement

User can sort products by price and name.

# Acceptance Criteria

1. Sort control provides expected options.
2. Sorting by price low-to-high updates order correctly.
3. Sorting by price high-to-low updates order correctly.
4. Sorting by name updates order alphabetically.
